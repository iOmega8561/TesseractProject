Tesseract

Download & Setup

  1. Download the latest binary release using convenient link

    You will be redirected to a GitHub release page. Under Assets you'll need to click on the line that ends with file extension .dmg.

    Your browser might ask you if you want to allow downloads from this website. Click on "Allow"

  2. Double click the file you just downloaded

    A window will open, containing the Tesseract application bundle, and a link to your Applications folder.

  3. Drag and drop Tesseract into Applications
  4. Start Tesseract from your Mac's launchpad

    An error will appear, because you downloaded the application from outside the App Store, that is normal so don't panic.

  5. Click on "Show in Finder"

    Right click on Tesseract and select "Open"

  6. Confirm when the alert pops up

    The next time you try to open the application, it will not display an error.
